## Git commands sheet

**Git initialization**\
git init 

**git status check**\
git status

**stage a file change**\
git add <FILENAME>

**stage every file change**\
git add .

**commit staged files**\
git commit -m 'COMMIT_MESSAGE'

**push commited files to Github**
git push

**FIRST time Github repo push**\
git remote add origin <REPO_URL> \
git branch -M main  \
git push -u origin main  \

**existing repo clone**\
git clone <REPO-URL>

**Pull latest changes from remote repo AND updates local repo (Combination of fetch and merge)**\
git pull
